# my-tools

个人工具箱集合，包含各种实用脚本和小工具。

## 工具列表

### 集合页文案生成
- **collection-copy-tool** - AI 驱动的产品集合页文案生成工具

### 链接检查工具
- **301-404-checker** - 301 重定向和 404 死链检测
- **404-checker** - 完整功能的 404 链接检测工具
- **404-checker-simple** - 简化版 404 检测
- **404-get** - 404 页面获取工具
- **internal-link-checker** - 网站内链检查工具

### 文章提取工具
- **article-extract-cli** - 命令行文章提取工具
- **bulk-article-extractor-web** - 批量文章提取 Web 版

### Shopify 相关
- **shopify-scraper** - Shopify 店铺爬虫
- **shopify-blog-exporter** - Shopify 博客导出工具
- **shopify-blog-tool** - Shopify 博客工具

### 其他工具
- **translator** - 翻译工具
- **url-analyzer** - URL 分析工具
- **design-md-tmp** - 设计模板

## 目录结构

```
my-tools/

├── 404-checker-simple/        # 简化版404检查
├── article-extract-cli/       # 文章提取CLI
├── bulk-article-extractor-web/  # 批量文章提取web版
├── internal-link-checker/     # 内链检查
├── shopify-scraper/           # Shopify爬虫
├── shopify_blog_exporter/     # Shopify博客导出
├── translator/                # 翻译工具
├── url-analyzer/              # URL分析
└── README.md                  # 本文件
```
