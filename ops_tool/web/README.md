﻿# Ops Tool Web

这是 `ops_tool` 的本地网页版本。

## 作用

- 在页面中选择一个或多个 Word 文件。
- 输入网站域名并解析 sitemap。
- 可选上传关键词配置文件。
- 生成文章处理 Prompt 和 TD Prompt。
- 可选使用 OpenAI 兼容 API 直接生成结果。
- 查看每个任务的日志、结果和提示词。
- 一键复制需要的结果。
- 手动删除已完成任务。

## 启动方式

```powershell
cd D:\my-tools\ops_tool\web
npm run dev
```

启动成功后访问：

```text
http://127.0.0.1:3210
```

## 使用方式

1. 输入网站域名。
2. 选择 CMS 模式。
3. 选择一个或多个 `.docx` Word 文件。
4. 按需选择关键词配置文件。
5. 选择“仅生成 Prompt”或“API 直接生成”。
6. 点击开始任务。
7. 在任务详情中查看日志、Prompt、API 结果和 TD 结果。
8. 使用页面上的复制按钮复制需要的内容。

## 常见情况

- 如果再次运行 `npm run dev` 提示端口 `3210` 被占用，通常表示工具已经启动。
- 浏览器访问 `http://127.0.0.1:3210` 即可继续使用。
- API Key 会保存在当前浏览器本地，方便下次使用。
