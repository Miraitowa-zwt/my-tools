====================================
多语言翻译器 - 使用指南
====================================

【第一次使用】

1. 获取 API Key
   - 访问：https://aistudio.google.com/apikey
   - 登录 Google 账号
   - 点击 "Create API Key"
   - 复制生成的 API Key（以 AIza 开头）

2. 配置 API Key
   - 打开 index.html
   - 点击右上角 ⚙️ 设置
   - 粘贴 API Key
   - 点击保存

3. 开始翻译
   - 输入要翻译的内容
   - 选择源语言和目标语言
   - 点击 "翻译" 或按 Ctrl+Enter


【网络问题解决方案】

如果显示 "网络连接失败"：

方法 1: 使用代理/VPN
- 开启代理工具
- 确保能访问 google.com
- 刷新页面重试

方法 2: 检查代理设置
- Windows: 设置 > 网络和 Internet > 代理
- Mac: 系统偏好设置 > 网络 > 高级 > 代理
- 确保代理已启用

方法 3: 使用浏览器代理插件
- Chrome: SwitchyOmega
- Firefox: FoxyProxy
- 配置规则访问 Google


【常见问题】

Q: "API Key 无效"
A: 
- 确认 API Key 完整复制（没有空格）
- 确认来自 aistudio.google.com
- 重新生成新的 API Key

Q: "请求过于频繁"
A:
- 等待 1-2 分钟
- 不要快速连续点击翻译
- 免费配额约 15 次/分钟

Q: "连接超时"
A:
- 检查网络是否连通
- 确认代理是否工作
- 尝试刷新页面

Q: 翻译结果空白
A:
- 检查 API Key 是否有效
- 检查网络连接
- 查看浏览器控制台错误（F12）


【快捷键】

Ctrl + Enter  - 翻译
Esc          - 清空输入


【支持语言】

中文（简体/繁体）、英语、日语、韩语、
西班牙语、法语、德语、意大利语、
葡萄牙语、俄语、阿拉伯语、印地语


【技术信息】

模型：gemini-1.5-flash
API:  Google Gemini API
存储：本地浏览器（localStorage）


【故障排查】

1. 按 F12 打开浏览器控制台
2. 切换到 Console 标签
3. 点击翻译按钮
4. 查看错误信息
5. 根据错误信息解决

常见错误：
- Failed to fetch: 网络问题，需要代理
- 429: 请求太频繁，等待片刻
- 400: API Key 无效，重新配置
- timeout: 网络超时，检查连接


【获取帮助】

如果以上方法都无法解决：
1. 截图错误信息
2. 记录浏览器版本
3. 说明网络环境（是否使用代理）
4. 联系技术支持
