# 多语言翻译器 - Deep Translate

简洁、高效的多语言翻译工具，基于 Google Gemini AI。

## ✨ 特性

- 🌐 **13 种语言** - 中/英/日/韩/法/德/西等
- 🚀 **快速翻译** - Gemini 2.0 Flash 驱动
- 💾 **本地缓存** - 相同内容不重复请求
- 🌙 **深色模式** - 护眼主题切换
- 📋 **一键复制** - 快速复制译文
- 📜 **历史记录** - 自动保存最近 10 条
- ⌨️ **快捷键** - Ctrl+Enter 翻译，Esc 清空
- 🔒 **隐私保护** - API Key 仅存本地

## 🚀 使用

1. **打开文件** - 用浏览器打开 `index.html`
2. **配置 API Key** - 点击右上角 ⚙️，输入 Gemini API Key
3. **开始翻译** - 输入文本，点击翻译或按 Ctrl+Enter

### 获取 API Key

访问 https://aistudio.google.com/apikey 创建免费 API Key

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Enter` | 翻译 |
| `Esc` | 清空输入 |

## ⚠️ 常见问题

### 翻译失败

| 错误 | 原因 | 解决 |
|------|------|------|
| API Key 无效 | Key 错误或过期 | 重新获取并保存 |
| 网络连接失败 | 无法访问 Google | 检查网络/代理 |
| 请求过于频繁 | API 限流 | 等待 1-2 分钟 |
| 翻译超时 | 网络慢 | 检查网络连接 |

### API 限流

Gemini 2.0 Flash 免费配额：
- 每分钟约 15-60 次请求
- 每天约 1000-1500 次请求

**建议**：不要快速连续点击翻译
